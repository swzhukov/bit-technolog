## ЧАСТЬ 1. ПРОДУКТ

### 1.1. Что это

**AI-помощник технолога для ускорения создания техкарт в 1С:ERP** для мелкосерийного машиностроения с индивидуальными заказами.

**НЕ:**
- ❌ AI-надстройка для КИВТ
- ❌ AI-ассистент внедрения 1С:ERP
- ✅ Помощник технолога (именно)

### 1.2. Пилотный клиент

**ООО «ПК Техинком-Центр»** (г. Тверь) — производитель пожарной спецтехники (АЦ, ПСС, МТБ, УМП) на шасси КАМАЗ/ГАЗ/УРАЛ/МАЗ/ЗИЛ.

### 1.3. MVP v0.1 — архитектура

```
Конструктор (КОМПАС-3D)
  └─ Сохраняет .m3d/.cdw в watch-папку
       │
       ▼
Python Watcher (FastAPI)
  ├─ Парсит свойства детали (масса, материал, ГОСТ, габариты)
  └─ Передаёт в LLM-агент
       │
       ▼
LLM-агент (Claude Sonnet 4.5)
  ├─ Получает: свойства + аналоги из RAG + справочник оборудования + правила технолога
  ├─ Генерирует: черновик техкарты в JSON (6 вкладок)
  ├─ Считает: время операции
  └─ Выбирает: оборудование из парка предприятия
       │
       ▼
Web-интерфейс (HTMX + FastAPI)
  ├─ Технолог: список деталей, карточка, 6 вкладок
  └─ Действия: Утвердить / Отклонить / В 1С / Обновить
       │
       ▼
1С:ERP 2.5 — экспорт в Excel (v0.1) → JSON (v0.2)
```

**Стек:** Python 3.11+, FastAPI, SQLite, HTMX, Docker Compose. On-premise. Бюджет API: 20К₽/мес (≈ 7000 вызовов Claude Sonnet 4.5).

## ЧАСТЬ 4. КОНКУРЕНТЫ

### 4.1. ИИ-Технолог (tab-is.ru) — главный

**Что умеет:**
- Распознавание чертежа (PDF/TIFF/JPEG, **НЕ 3D**)
- Нормы по ГОСТ + маршрут + спецификация + техкарта + себестоимость + металлоёмкость
- Аналоги материалов
- Проверка пригодности оборудования
- **Архитектура:** оркестр моделей + роботы (китайские коммерческие + другие)

**Цены (июль 2026):**
- ППО: **180 000 ₽** фикс
- Лицензия: от **300 000 ₽** (каталог) до **1 100 000 ₽** (Профессионал), от 3 млн (Интерпрайс)
- Поддержка: 100К/год
- Внедрение: **может быть > лицензии** (1,5М лиц → 1,7-2М внедрение)
- Железо: 0,5-1М ₽
- Итого вход: 1,3-2,5М+

**Точность:** 88-95% после обучения, 40-60% без
**Скорость:** 1-5 мин/чертёж
**Среда:** АстроЛинукс ОК, Windows «печально»
**On-premise:** да, может без интернета

**Известный кейс:** пожарные машины в ГОЗ-закупках, каждая индивидуально. Совпадение с Техинкомом высокое, но **прямого упоминания нет** (NDA).

- **Вертикаль спецтехники** (не горизонтальный AI)

## ЧАСТЬ 7. СТИЛЬ РАБОТЫ С СЕРГЕЕМ

### Что делать
- Префикс **«ты — лучший в мире [роль]»** для промтов
- «Покритикуй», «задавай вопросы» — давать другой взгляд
- Длинные структурированные промты с контекстом — нормально
- Спорить и давать прямой взгляд — он настаивает
- Понять **главную цель** (а не завуалированную)
- При длинном контексте — **фактология > гипотезы**
- **«Ты — лучший в мире»** для промтов

### Чего НЕ делать
- ❌ Не уходить в сторону (юр/Минпромторг — мимо)
- ❌ Не делать широкие обзоры без запроса
- ❌ «Отличный вопрос», «спасибо за уточнение»
- ❌ Длинные вступления и «прежде всего»
- ❌ Додумывать имена людей без grep'а
- ❌ Распыляться на уточнения, не влияющие на продукт
- ❌ Не врать про усилия
- ❌ Не задавать вопросы, на которые можно ответить самому

### Формат ответов
- Структурированный вывод (таблицы, карточки)
- Конкретные рекомендации с обоснованием
- Без hedging
- Артефакты через `<deliver-assets>`
- Файлы в `/workspace/`
