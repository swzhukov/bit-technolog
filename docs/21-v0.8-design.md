# Дизайн-система v0.8 — БИТ.Технолог

> **Концепция:** B2B SaaS (Linear / Vercel / Notion style). Тёмный header, светлый body. **Красный — только для primary CTA и критичных бейджей.** Минимализм, generous whitespace, тонкие границы.

## Принципы

1. **Минимализм** — generous whitespace, не больше 2 цветов на экране кроме серых
2. **Красный — sparingly** — только для primary CTA, активных пунктов nav, и важных бейджей (новое извещение, конфликт)
3. **Тёмный header** — пользователь чётко знает где top, светлый body не давит на глаза
4. **Системные шрифты** — Inter, fallback на system-ui (быстро, знакомо)
5. **Одна палитра серых** — текст, муть, финт, граница — все из одной шкалы Tailwind
6. **Никаких emoji-иконок в UI** — только в content (🔥, 📄, 📬) для удобства чтения
7. **Inline-styles запрещены** — только CSS-классы из design-system
8. **Hover/active состояния** на всех интерактивных элементах

## Что НЕ делаем

- ❌ Тени (box-shadow) больше 1px — выглядит дёшево
- ❌ Градиенты — выглядит как 2010
- ❌ Скругления больше 12px — выглядит как мобильное приложение
- ❌ Кастомные шрифты с CDN — увеличивает TTFB
- ❌ Анимации > 200ms — раздражает
- ❌ Модальные окна — вместо них inline panels
- ❌ Тёмная тема body (только header) — утомляет глаза
