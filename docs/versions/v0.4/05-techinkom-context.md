### 📄 DOCX 13bd245e — **«Структура предприятия + Маршрутные листы»** (ЗОЛОТО)
- **Что:** Полная структура + маршруты изготовления УМК и АЦ 6,0-40
- **Структура предприятия:**

```
Участок обработки металла (Заготовительное производство)
├── Уч-к лазерной резки
├── Уч-к механической резки (гильотина)
├── Уч-к гибки
└── Уч-к механообработки

Мелкосерийное производство
├── Уч-к изготовления упоров
├── Уч-к изготовления растяжек
└── Уч-к комплектования и упаковки

Производство высотной техники (ПСС)
├── Сварочно-сборочный уч-к ВТ
└── Сборочный уч-к ВТ

Производство кузовной техники (АЦ)
├── Сварочно-сборочный уч-к КТ
└── Сборочный уч-к КТ
```

- **Маршрут УМК:**
  - Лазерная резка → Токарная (опционально) → Сборка/прихватка продольных упоров → Сварка → Зачистка → Бандажи → Ключи → Сварка растяжек → Сборка узлов растяжек → Зачистка → Сборка полуфабриката → Комплектование → Упаковка → Установка бандажей для отправки на КАМАЗ
- **Маршрут АЦ 6,0-40 (КУЗОВ):**
  - Уч-к обработки металла: Резка лазер / гильотина / гибка / токарная / фрезерная
  - Сварочно-сборочный уч-к КТ: Сборка цистерны → Сварка → Сборка каркаса КБР → Сварка КБР → Сборка отсеков → Сварка отсеков → Подготовка шасси → Установка цистерны, отсеков, пенобака, КБР → Установка гидрооборудования
  - Сборочный уч-к КТ: Сборка гидрооборудования → Электромонтаж → Сборка отсеков и КБР → Установка ПТВ
- **Для нас:** 🟢 **КАРТА ЦЕХОВ И МАРШРУТОВ** — то, что мы кладём в промт и в RAG
