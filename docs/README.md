## Быстрый старт (TL;DR)

**Что это:** прототип AI-помощника, который по свойствам детали от конструктора (из КОМПАС-3D или 1С) генерирует черновик технологической карты.

**Чем отличается от аналогов:**
- Стоимость входа: **0₽** (vs ИИ-Технолог 1.1М₽)
- Срок запуска: **2-4 недели** пилота (vs 6-12 месяцев)
- 100% on-prem (ГОЗ, оборонка)
- 3-step UX: уточнения → draft → полная ТК

**Что работает (87/87 тестов):**
- Генерация черновика ТК с операциями, временем, оборудованием
- RAG на 50-100 исторических ТК (TF-IDF, on-prem)
- 2-3 альтернативных маршрута
- Inline-редактирование операций
- Печатная форма ТК (A4, с подписями, QR)
- CSV-экспорт для 1С:ERP
- Audit log + экспорт для ISO 27001
- Ролевая модель (draft → review → approved_chief)

**Что НЕ работает / отложено:**
- Прямая интеграция с 1С:ERP (Sprint 4 — отдельный проект, 1.8М₽, 6-8 нед)
- КОМПАС-3D Watcher (отложен)
- Docker on-prem (Sprint 5)

**Готовность к пилоту: 85%.** Блокирующее: данные Техинкома (50-100 исторических ТК) + ответы на 9 вопросов в `09-open-questions.md`.
